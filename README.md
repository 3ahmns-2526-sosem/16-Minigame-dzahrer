# \# Clicker Game

# 

# \## Spielidee

# 

# In diesem Mini-Game muss der Spieler auf ein Objekt klicken, um Punkte zu sammeln.

# 

# Das Ziel ist es, 10 Punkte zu erreichen.

# 

# \## Steuerung

# 

# \* Linksklick auf das Quadrat

# \* Jeder Klick erhöht den Score um 1

# 

# \## Ziel

# 

# Erreiche 10 Punkte.

# 

# \## Feedback

# 

# \* Der aktuelle Score wird angezeigt.

# \* Bei 10 Punkten erscheint die Meldung "YOU WIN!".

# \* Danach sind keine weiteren Punkte mehr möglich.
